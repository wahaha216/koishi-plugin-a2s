# @wahaha216/koishi-plugin-a2s

[![npm](https://img.shields.io/npm/v/@wahaha216/koishi-plugin-a2s?style=flat-square)](https://www.npmjs.com/package/@wahaha216/koishi-plugin-a2s)

使用 A2S 协议获取起源引擎游戏信息

## 使用方法

```tex
a2s <ip>
```

开启自动识别后，将识别以下格式

```tex
connect <ip>
```
